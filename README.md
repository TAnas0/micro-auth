# micro-auth
Authentication microservice

### run
`$ npm run start:dev`

### lint
`$ npm run lint`

### migrate
`$ npm run migrate-db`

### seed data
`$ npm run seed-db`

### setup
```
$ npm install
$ npm run migrate-db
$ npm run seed-db
$ npm run start:dev
```
